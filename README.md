# 🎬 FreeTube Inspired Landing Page

## 📌 Overview
This project is a modern and responsive **software landing page UI** inspired by FreeTube.

It is built using HTML, CSS, and JavaScript and showcases a real-world product website structure including features, screenshots, download sections, and useful links.

---

## 🚀 Features

- 🎯 Modern responsive UI design
- 🧩 Features showcase grid
- 🖼️ Product screenshots section
- 💻 Multi-platform download section (Windows, Mac, Linux)
- 🔗 Useful links (GitHub, Docs, Blog, Translation)
- 💰 Donation section UI
- 📱 Mobile-friendly layout
- 🎨 Clean and structured design

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
- Font Awesome Icons
